## ✨ 简介

Bingo Agent Hub 是一个统一的 AI Agent 平台入口，整合了三个独立的 AI Agent 项目：

- 📊 **AIAgentDevOps** - DevOps 仪表板
- 🧠 **Bingo AI** - AI 工程模板
- 🎯 **Bingo Skills** - 技能库

## 🚀 快速开始

### 访问平台

- **在线访问**: https://bingo-agent-hub.vercel.app
- **GitHub**: https://github.com/jiangbingo/bingo-agent-hub

### 本地运行

```bash
# 克隆仓库
git clone https://github.com/jiangbingo/bingo-agent-hub.git
cd bingo-agent-hub

# 打开 index.html
open index.html
```
